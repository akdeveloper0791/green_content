IOT device registration API:

Iot Device Registration request takes input of user login info and device info, and returns device regisration key. The user info is like user name password and device info as device mac address, device name.

IOT device registration request is an HTTP URL of the following form:

	https:https://www.signageserv.ai/iot_device/register_as_third_party/

where output may be the following values:

  • json (recommended) indicates output in JavaScript Object Notation (JSON)

Certain parameters are required to initiate a Device Registration request. As is standard in URLs.

Required parameters

  • user_email —The text input specifying which user account and that is your account Login Email id. This value identifies your account. See Register page for more information.
  • pwd — The text input specifying which is user account password.
  • data — The type of input is JSON Object as a string field. This contains following parameters.
    • name — The text input specifying which is IOT device name.
    • mac — The text input specifying which is IOT device mac address.
