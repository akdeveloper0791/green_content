Iot device registration API:

Iot Device Registration request takes input of user login info and device info, and returns device regisration key. The user info is like user name password and device info as device mac address, device name..

A Find Place request is an HTTP URL of the following form:

	https:https://www.signageserv.ai/iot_device/register_as_third_party/
